# Dangam Market
Ruby on Rails를 학습한 내용들을 직접 적용해 보기 위해 만든 단순 CRUD 프로젝트입니다. <br/> 
Ruby와 Rails를 공부한 내용들은 **[[공부한 내용 정리 페이지 바로가기]](https://github.com/binary-ho/Ruby-on-Rails)** 이곳에 정리해 두었습니다.
